# Notadd Imagemanager

ImageManager 是一个图片上传及管理插件。

## 安装

先在前端目录下：/images-manager/resources/mixes/administration	  
运行npm install 来安装必须文件	    
然后npm run build来打包压缩文件到dist目录	    
最后在后台安装和启用就可以了	    

## 访问图片 
需要将storage文件夹下的uploads/images 加软链或快捷方式到 public/uploads/images 下
